# Agent Notes

This folder stores guidance for auxiliary agents or delegated work.

Any agent working on this repository should:

- Read `CODEX_CONTEXT.md` before making changes.
- Keep user-facing application text in English.
- Explain work to the user in Portuguese.
- Treat `main.py` as the main Tkinter application surface.
- Treat `app/signal_processing.py` as the signal parsing, synchronization, scalar calculation, and processing layer.
- Preserve compatibility with saved validation projects.
- Prefer small, focused changes and verify imports or syntax after edits.
